#  JAVA SCRIPT
### JAVA와 Html의 연결
* script 태그 사이에 작성

### 이벤트
* alert() : 경고문
* .length : 글자 수 알려줌
* onclick : 클릭 시 실행하게 하는 속성
* onchange : 입력값이 달라졌을 때 실행됨
* onkeydown : 키를 눌렀을 때 실행됨

### 콘솔
>파일 외 환경에서 간단히 자바 실행 가능
>elements에서 esc누를 시 콘솔 분할 화면 가능

### 데이터 타입
* boolean : T/F
* null
* undefined
* number : 연산
* string
* symbol

### 리팩토링
* this : 자기 자신을 가리킴(인덱스 태그를 간결하게 할 수 있음)
* var target = ~~ : 중복되는 타켓 정리가능

### 배열&반복문
* 배열 : 수많은 데이터 정리정돈을 위함(파이썬 리스트형식과 유사)
 * var 변수 = ["값", '값2"];
 * 변수[0] : 0번째 꺼내오기
 * 변수.length : 길이
 * 변수.push("값") : 값 추가

* 반복문 : 
 * ''' while(T/F){실행문}
       조건 불일치 시 실행문 ''' : 기본 문법
